Methods

scan(args, cb)

Treeman's main function, scans through a folder down the tree. Applies a callback function to all matching files inside the directory tree.

Parameters:
Name Type Description
args Object

Arguments containing instructions for the tree scan method.

Properties
Name Type Attributes Description
dirPath string

The path to your folder tree root.
ignoreList Array.<string> <optional>

Optional list of files/folder names/extensions to ignore. example: ['node_modules','.css'] will ignore every file inside node_modules, as well as every css file in your tree.
cb fileCallback

A callback that will apply to every file found in the scan.
